# Databases

## Build database
- Have Docker installed and running
- Go to database directory, for example: `cd postgresql`
- Run: `docker-compose up -d`

## Credentials

### PosgreSQL
- User: `postgres`
- Pass: `example`

### MySQL
- User: `root`
- Pass: `example`

### MariaDB
- User: `root`
- Pass: `example`
